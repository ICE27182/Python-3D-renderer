## How to render?
### What features should my renderer possess?
- Workability
- Able to read .obj file
- Support parallel light
- Z-buffer
- Rasterization
### What other features does my renderer possess?
- Able to manipulate models to some extent
### What features can be added?
- UV Texture
- Smooth Shading
- Anti-aliasing (FXAA)
